Semana 3 de la misión backend con NodeJS.

El aprendizaje en esta semana fue muy productivo, aprendimos muchas cosas esenciales, como el directorio de un proyecto en javascript, utilizando las carpetas app, test y sus respetivos subdirectorios como modules, views y services, también la sintaxis de una prueba de unidad con Jest, como ejecutar el proyecto (index.js) con npm modificando el package.json (especificamente el apartado de “scripts”), también como correr las pruebas de unidad y donde modifiar el package.json para trabajar con Jest en ellas. Así mismo se elaboró un proyecto de una API con express, lo cuál fue mi primer acercamiento a la elaboración de una y se me hace un suceso importante, puesto que leemos, escuchamos e incluso consumimos las APIs (he utilizado la de Google Drie para PHP y Google Calendar también para PHP) todos los días y nuna había elaborado una propia.
